## 기본 조작법
- 이동: 방향키
- 확인 : Z
- 취소 : X
- 메뉴창 : Enter

## 플레이 편의 기능
- 스페이스바 : 2배속
- 데미지 배율 변경 : Q
- 경험치 획득 배율 변경 : W
- 야생 포켓몬을 만날 확률 변경 : E
- 포획률 변경 : R
- 배율 확인 : F2
- 충돌맵 켜고 끄기 : O

## 맵 이동 기능
### (주의) 포켓몬을 데리고 있지 않은 상태로 다른 마을로 이동한 뒤 수풀에 들어가면 버그가 발생할 수 있습니다.
- <태초 마을>로 이동 : 1
- <상록 시티>로 이동 : 2
- <상록숲>으로 이동 : 3
- <회색 시티>로 이동 : 4
